# Sistema de Gerenciamento de Restaurantes

Projeto em Python para estudar Programação Orientada a Objetos.

## Funcionalidades

- Cadastro de restaurantes
- Ativação/desativação
- Registro de avaliações
- Cálculo automático de média
- Listagem formatada

## Conceitos aplicados

- Classes
- Encapsulamento
- Propriedades
- Métodos de classe
- Organização em módulos
